Projects What I Tested Two real products, tested across multiple stages. Project 1 SkillBridge Staging / Partial Production SkillBridge is a talent pipeline platform that connects candidates with employers through verified skill assessments. I was the only QA engineer on the team. I built and ran an automated API regression suite of 78 requests across 11 modules using Postman and Newman CLI. This covered auth, session handling, settings, assessments, and role-based access with positive and negative test cases. I also did a manual test pass on the Verified Profile feature against an 11-point acceptance charter, logged defects with severity and reproduction steps, and tested accessibility and API security in later stages. Postman Newman CLI k6 Lighthouse API Testing Accessibility Testing Performance Testing OWASP Top 10 WCAG Project 2 ClinSight MVP Stage ClinSight is a healthcare AI platform. I joined during the MVP stage and focused on the authentication flow and OTP verification. I wrote a full QA test plan covering user signup, OTP verification, password reset, session validation, and API error handling. The plan included entry criteria, exit criteria, a risk priority framework, and boundary testing for OTP expiration and repeated failed attempts.
